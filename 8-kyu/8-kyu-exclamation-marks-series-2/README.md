# Exclamation marks series #2: Remove all exclamation marks from the end of the sentence

### Description:

Remove all exclamation marks from the end of sentence.

Examples
```
"Hi!"     ---> "Hi"
"Hi!!!"   ---> "Hi"
"!Hi"     ---> "!Hi"
"!Hi!"    ---> "!Hi"
"Hi! Hi!" ---> "Hi! Hi"
"Hi"      ---> "Hi"
```
